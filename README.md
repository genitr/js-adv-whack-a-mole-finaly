## Ударь гоблина!

### Правила игры:
- Гоблин появляется в случайной точке ровно на 1 секунду. 
- Если игрок успел за это время кликнуть на этой точке, то засчитывается +1 балл.
- Если игрок пропустил 5 появлений гоблинов, то игра завершается.
